Mulai dari CSV

Unggah CSV hasil dari Google Sheets/Excel. Program akan otomatis membuat grouping DPL beserta mahasiswa bimbingannya. Kolom minimal: DPL, NIM, Nama. Kolom link berkas bersifat opsional.

Format CSV disarankan: dpl, nim, nama, kelas, sekolah, link_rpp, link_media, link_alat_evaluasi, link_bahan_ajar, link_laporan_akhir

Pilih DPL, lalu pilih mahasiswa. Nilai lima komponen berkas menggunakan skala 0–100. Nilai akhir dihitung dari rata-rata lima komponen, kecuali bobot diubah.

Rekap Penilaian

Rekap ini dapat diekspor CSV. Jika ingin tersimpan ke Google Sheets, isi URL Apps Script pada menu Pengaturan, lalu klik kirim saat menyimpan nilai.

Pengaturan Simpan ke Google Sheets

Web ini tetap statis. Agar nilai bisa masuk Google Sheets, gunakan Google Apps Script sebagai jembatan. Jika URL Apps Script dikosongkan, nilai hanya tersimpan di browser dan bisa diekspor CSV.

Mode Simpan Nilai

Saran awal: gunakan Simpan lokal saja saat uji coba. Setelah Apps Script siap, ubah ke Simpan lokal + kirim ke Google Sheets.

Kode Apps Script

File ZIP menyertakan apps_script.gs. Kode itu ditempel di Google Apps Script, lalu deploy sebagai Web App.

Password Kaprodi

Password ini dipakai untuk masuk sebagai Kaprodi dan melihat seluruh grouping serta semua password DPL. Segera ganti password bawaan.

Akun & Kontrol DPL

Setiap DPL punya password sendiri (dibuat otomatis dari data CSV, bisa Anda ubah). Tabel ini juga memantau progres pengisian nilai tiap DPL.
